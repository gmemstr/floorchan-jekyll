General rules (applies to all services and aspects of Floorchan)

  • Don't touch stuff that doesn't belong to you.
  • Don't be mean to other users.
  • Don't try to hack into other users' accounts.
  • Don't purposefully try to crash the server.
  • Be considerate of others.

SSH rules

  • Please be considerate when running tasks that may consume large amounts of CPU, RAM, Network, or Disk I/O.
  • You are responsible for backing up all of your files. Floorchan is not responsible for loss of data in the event of human error, hardware/software failure, or factors beyond our own control.
  • While we always try to maintain the highest level of security possible, please be aware that Floorchan is shared between many users and any and all content that you upload should be considered public.
  • All content that you upload to our server is always your property.
  • If you wish to use your account to run cronjobs, eggdrop scripts, or application servers, please contact camoyoshi@trioptimum.com first. Failure to do so will cause your account to be instantly deleted.
  • You may store up to 5GB of data in your /home directory.

FTP usage

  • You have a public FTP folder located at ~/public_ftp/. Placing files into this area will publish them via FTP and HTTP, at the following addresses: ftp://ftp.floorchan.org/users/yourname/ & https://ftp.floorchan.org/users/yourname/
  • We use proftpd for our FTP daemon.
  • Our FTP server only accepts anonymous connections and will not accept uploads. If you wish to upload files use SFTP (sometimes called SSH FTP or FTP over SSH) instead.

WWW Userpages usage

  • You have a public HTTP folder located at ~/public_html/. Placing files into this area will publish them via HTTP only, at the address (https://users.floorchan.org/~yourname).
  • For our HTTP daemon, we use Apache2 with PHP & CGI.
  • If your website uses any PHP or CGI, please be considerate and only run safe functions (we have safe_mode = on at all times).
  • Don't run scripts that use lots of CPU or RAM, be considerate.
  • In order to prevent such situations, the following software is banned:
    • Wordpress
    • PhpMyAdmin
    • Concrete5

Server Restrictions

  • Maximum number of simultaneous logins: 2
  • Maximum number of simultaneous processes: 100
  • Maximum amount of allocatable memory: 4GB
