Floorchan initially started as a free webhost run by ThatSandersKid in 2011. Shortly after opening, Floorchan laid silent for a few years until recently, it became the website you are viewing now.

Floorchan is simply put, a server hosted in Atlanta GA, but the main guy who runs it is from Seattle, WA. We provide network services to the general public.
